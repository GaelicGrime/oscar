Bem-vindo ao OSCAR software de código aberto para análise de dados CPAP


Este software foi desenvolvido para auxiliar você na revisão dos dados produzidos pelo seu aparelho CPAP.

POR FAVOR, LEIA COM ATENÇÃO

OSCAR não é um substituto do acompanhamento feito por um profissional.

Devido à falta de documentação disponibilizada pelos fabricantes de aparelhos CPAP, a cerca dos dados gravados pelos aparelhos, a precisão das informações exibidas no OSCAR não podem ser sempre garantidas.

Todos os relatórios gerados são para USO PESSOAL, e foram feitos para serem tão precisos quanto possível.

Os relatórios do OSCAR são baseados nos dados do seu aparelho CPAP. A aprovação desses dados, para quaisquer fins, está sujeita às unidades competentes.

O uso deste software é por sua conta e risco. Os autores não serão culpados por nenhum mal uso relacionado à este software.
