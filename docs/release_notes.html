SleepyHead v0.9.0 BETA

Release Notes

Beta.. Hooray.. I know it doesn't feel much different :)

What does this mean for SleepyHead you ask? It means now the primary focus has been shifted away from design to smashing bugs.

SleepyHead's database (SleepLib) has had it's final work done on it, and is officially locked. Any changes made from now on will require a full upgrade path.

Unfortunately it means starting fresh for this series, however the good news is this version can coexist with the old one while you copy your notes over.

There is still some stuff to do, but apart from bug-fixing, it's mostly cosmetic and easy stuff from now on in.

New Features:

  • Auto-Updater for Windows & Mac Platforms
  • SleepLib database improvements, and lots of other underneath stuff and code cleanups you don't see.
  • New Summary page showing some useful text statistics.
  • Welcome page has become a Help Browser.
  • New Navigation Panel on the right side, now accessible from all tabs. It can be hidden to reclaim screen space.
  • New Favourites tab in this right panel for bookmarking days and quickly finding them again.
  • New Records tab to show some best/worst entries, which have links going directly to the related day or overview range.
  • Daily reports bookmarked areas show oximetry data when available.
  • Can print from both the Summary & Help Browser pages.
  • New Context cube can be switched on to make empty pages more attractive.
  • Plenty of other bug fixes, including more oximetry fixes.

    • Important Information:

  • Just to repeat, SleepyHead can't work with the older database format. I tried to make it update, but it didn't go to plan. Sorry.
  • SleepApp has now been renamed to SleepyHeadData, it still resides in your Documents folder so it's easy to find.

    • Planned Improvements for this beta series:
    The hard stuffs done, and everything needs time to settle and have the bugs knocked out of it, however there still are some minor tweaks and features in the pipeline...

  • Lots more Performance Optimizations
  • Graphical tweaks, animations & improvements
  • Improving the quality of printed report data, and showing more data and statistics.
  • Improving the CSV export feature, and making it more customizable
    •  But mostly we are talking serious bug smashing time!!!
